Run the Pitch Deck exercise from `~/.claude/skills/pitch/SKILL.md` (Stage 1: Pitch Deck).

1. Ask the user for: company name, one-line description, stage (pre-seed/seed/Series A), and current traction metrics
2. Walk through all 12 slides (Title through The Ask) using the Sequoia framework
3. For each slide, generate specific content tailored to their product — not generic templates
4. Flag any anti-patterns from the Deck Anti-Patterns checklist
5. Output a complete slide-by-slide outline ready to build in Keynote/Google Slides/Figma
6. Highlight which slides are strongest and which need more data before pitching
