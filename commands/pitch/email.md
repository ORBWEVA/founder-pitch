Run the Investor Outreach exercise from `~/.claude/skills/pitch/SKILL.md` (Stage 3: Investor Outreach).

1. Ask the user for: company name, one-line traction hook, round details, and whether they have warm intros available
2. Generate a cold VC email with a specific subject line and traction proof point
3. Generate a warm intro request email with a forwardable blurb
4. Generate a follow-up sequence (Day 5-7 and Day 12-14) — each adding new information
5. Build an investor targeting framework with Tier 1/2/3 prioritization
6. Score each email draft on personalization level (1-5) and suggest improvements
7. Remind: run 10-15 meetings in 2-3 weeks — create urgency, not a drip campaign
