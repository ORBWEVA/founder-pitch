Run the Financial Model exercise from `~/.claude/skills/pitch/SKILL.md` (Stage 2: Financial Model).

1. Ask the user for: business model type, current MRR/revenue, ARPU, churn rate, growth rate, team size, and raise target
2. Build a 3-year revenue projection with explicit assumptions
3. Build a cost structure (COGS + OPEX) with realistic scaling
4. Create a use-of-funds bridge showing how the raise maps to milestones
5. Run a sensitivity analysis on the 3 most critical assumptions
6. Identify which assumption matters most — this is what VCs will challenge
7. Output all tables formatted for easy copy into a spreadsheet or deck appendix
